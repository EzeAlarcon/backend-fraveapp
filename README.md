# BACKEND - fraveapp

## NodeJs - Typescript - Mysql - Socket io

## Installation

---

### Mysql

- Go to Folder Database where you will the script for the creation of Database and the stored procedures


### Nodejs

```sh
    npm install
```

### Running the project

```sh
    npm start
```

### Delete  the Dist folder

```sh
    npm run clean
```

### Environment variables

- PORT
- TOKEN_SECRET
